Two Player Tic Tac Toe Using Socket Programming

Tic-Tac-Toe is a very popular game amongst people of all age groups requiring two players and a board. The objective of the game is to have three marks in a row: horizontally, vertically or diagonally as two players make marks on the board, one after the other.
In this implementation of Tic-Tac-Toe on screen, p5.js is used to make the game, and Socket.IO with node.js is used to establish connections between a server and two users. One user acts as Player O and the other as Player X, and both players make marks on the board upon their respective turns. As they do so, the server constantly receives data from both the users and broadcasts it to ensure both players have the updated version of the board. An additional functionality added to the implementation is the chatbox at the right, which the players can use to communicate with one another.

4 bit ALU

Features Implemented

  1. The project implements a 4-bit ALU.
  2. It is capable of performing addition, subtraction, and logical operations, namely, AND and OR on two 4-bit numbers.
Methodology
  1. Numbers are inputted via two 4-bit registers, A and B, which are implemented using D-flip flops.
  2. Use of four full adders for addition and subtraction.
  3. Use of 2x4 multiplexer for performing one of the four operations.
  4. The ALU performs the selected operation on the two numbers and feeds the result back into Register A, from where it can be loaded into the ALU again;
  5. The registers are clock-synchronized.

Unit Converter App made with Azure

Microsoft Azure is a popular cloud computing platform that offers a variety of services. To explore some of these services, we developed and deployed a scalable unit conversion application. The deployment of the backend uses Azure Function Apps, with the frontend stored in the Azure Blob Storage as a static webpage. The Azure App Service plays a key role in managing the execution environment of the function, and Azure Application Insights allows constant monitoring of the application.
